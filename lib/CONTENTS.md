# Shared libraries

## Why this folder exists

Pure utilities, clients, and helpers with no React views. Imported by features/ and app/.

## What is here

- catalog/ — static params, product merge, blocks2d
- auth/, supabase/ — client-safe auth helpers
- ai/, analytics/, tracking/
- hooks/, helpers/, types/, ui/ (non-component helpers)

## Ownership

| Area | Canonical owner | Typical consumers | Do not duplicate in |
|------|-----------------|-------------------|---------------------|
| Server session & route guards | `lib/auth/` | app/api, app/crm, app/admin, app/planner layouts | features/shared/auth/ — React providers and login UI |
| Catalog SSG & merge pipeline | `lib/catalog/` | features/catalog/getProducts, app/(site)/products | features/catalog/ — filter UX and domain types |
| Supabase browser client | `lib/supabase/` | features, components (client components) | platform/supabase/ — migrations, admin, edge functions |
| Site quote cart | `lib/store/` | marketing quote flows | features/planner/store/ — planner Zustand only |
